## 🔍 WordPressユーザー名の確認方法

WordPressのユーザー名は、WordPressにログインする際に使用する**ユーザー名**です。

### 確認方法（2通り）

#### 方法1: ログイン画面で確認
1. https://freeeeeeestyle.com/wp-admin/ を開く
2. ログイン画面の「ユーザー名」欄に入力している名前を確認
   - これがユーザー名です！

#### 方法2: WordPress管理画面で確認
1. WordPressにログイン
2. 右上のユーザー名をクリック
3. 「プロフィールを編集」をクリック
4. 「ユーザー名」の欄を確認（表示されているか、変更できない欄として表示されます）

---

## 📝 ユーザー名を設定する

`.env`ファイルを開いて、以下の行を修正してください：

```env
WORDPRESS_USERNAME=あなたのユーザー名をここに記入
```

**例：**
```env
WORDPRESS_USERNAME=admin
```
または
```env
WORDPRESS_USERNAME=keiji
```

### 2. 成功したら

### 3. 毎日自動投稿を開始する

設定が確認できたら、スケジューラーを起動します：

```bash
python main.py --scheduler
```

これで毎日指定した時刻（デフォルト: 09:00）に自動で記事が投稿されます。

---

## 🆘 エラーが出た場合

### エラー: "WORDPRESS_USERNAMEとWORDPRESS_PASSWORDを.envファイルに設定してください"
→ ユーザー名が正しく設定されているか確認してください

### エラー: "401 Unauthorized" または "認証エラー"
→ ユーザー名またはアプリケーションパスワードが間違っている可能性があります
→ `.env`ファイルの内容を再確認してください

### エラー: "OpenAI API キーエラー"
→ OpenAI APIキーが正しく設定されているか確認してください
→ APIキーに余分なスペースや改行が入っていないか確認してください

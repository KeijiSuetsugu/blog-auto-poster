# 🎯 超簡単！2つのキーを取得する方法

## 🔐 WordPressのアプリケーションパスワード

### やり方（3ステップ）

1. **WordPressにログイン**
   - https://freeeeeeestyle.com/wp-admin/ を開く
   - ユーザー名とパスワードでログイン

2. **プロフィールページを開く**
   - 左のメニューで「ユーザー」→「プロフィール」をクリック
   - または、右上のユーザー名をクリック→「プロフィールを編集」

3. **アプリケーションパスワードを作る**
   - ページを下にスクロール
   - 「アプリケーションパスワード」のセクションを見つける
   - 名前を入力（例：「自動投稿」）
   - 「新しいアプリケーションパスワードを追加」をクリック
   - **表示されたパスワードをコピー**（例：`abcd efgh ijkl mnop`）
   - ⚠️ このパスワードは一度しか表示されないので、必ずコピーすること！

## 🤖 OpenAI APIキー

### やり方（4ステップ）

1. **OpenAIのサイトを開く**
   - https://platform.openai.com/ を開く
   - ログイン（アカウントがない場合は作成）

2. **API Keysページを開く**
   - 右上の自分の名前/アイコンをクリック
   - 「API keys」をクリック
   - または、左のメニューから「API keys」を選択

3. **新しいキーを作る**
   - 「+ Create new secret key」をクリック
   - 名前を入力（例：「ブログ投稿用」）
   - 「Create secret key」をクリック

4. **キーをコピー**
   - **表示されたキーをコピー**（`sk-`で始まる長い文字列）
   - ⚠️ このキーも一度しか表示されないので、必ずコピーすること！

---

## 📝 .envファイルに貼り付ける

コピーした2つのパスワード/キーを、`.env`ファイルに貼り付けます：

```env
WORDPRESS_USERNAME=あなたのユーザー名
WORDPRESS_PASSWORD=コピーしたアプリケーションパスワード
OPENAI_API_KEY=コピーしたAPIキー
```

これで完了！

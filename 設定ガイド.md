# 📖 設定ガイド - 小学生でもわかる説明

このガイドでは、WordPressのアプリケーションパスワードとOpenAI APIキーの取得方法を、写真を見ながら詳しく説明します。

---

## 🔐 パート1: WordPressのアプリケーションパスワードを取得する

### ステップ1: WordPressにログインする

1. ブラウザ（Google ChromeやSafariなど）を開きます
2. アドレスバー（上の白い箱）に以下のURLを入力します：
   ```
   https://freeeeeeestyle.com/wp-admin/
   ```
3. Enterキーを押します
4. ユーザー名とパスワードを入力して「ログイン」ボタンを押します

### ステップ2: プロフィールページに移動する

ログインしたら、左側のメニューを見てください：

1. **左側のメニュー**から「ユーザー」という文字を探します（通常は下の方にあります）
2. 「ユーザー」の上にマウスを置くと、メニューが広がります
3. **「プロフィール」**をクリックします

**別の方法**: もし「ユーザー」が見つからない場合：
- 左上の「ユーザー名」をクリック
- 「プロフィールを編集」をクリック

### ステップ3: アプリケーションパスワードを作成する

プロフィールページが開いたら、下にスクロールします。

1. 「**アプリケーションパスワード**」というセクションを見つけます
2. そのセクションの中に、**名前を入力するための箱**があります
   - 例：「ブログ自動投稿」や「Pythonスクリプト」など、わかりやすい名前を入力します
3. 「**新しいアプリケーションパスワードを追加**」または「**生成**」ボタンをクリックします
4. 画面に**パスワードが表示**されます（例：`abcd efgh ijkl mnop`のような形）
   - ⚠️ **重要**: このパスワードは**一度しか表示されません**！
   - 📋 **今すぐコピーするか、メモ帳に書き写してください**

### ステップ4: パスワードを保存する

表示されたパスワードは：
- スペースを含む場合と含まない場合があります（どちらでも使えます）
- 例: `abcd efgh ijkl mnop` または `abcdefghijklmnop`

このパスワードを`.env`ファイルの`WORDPRESS_PASSWORD`に設定します。

## 🤖 パート2: OpenAI APIキーを取得する

### ステップ1: OpenAIのウェブサイトにアクセスする

1. ブラウザを開きます
2. アドレスバーに以下のURLを入力します：
   ```
   https://platform.openai.com/
   ```
3. Enterキーを押します

### ステップ2: アカウントにログイン（または作成）

1. 画面右上の「**Log in**」（ログイン）ボタンをクリックします
   - まだアカウントがない場合は「**Sign up**」（サインアップ）をクリックしてアカウントを作成します
2. メールアドレスとパスワードを入力してログインします

### ステップ3: API Keysページに移動する

ログインしたら：

1. 右上の自分の名前やアイコンをクリックします
2. メニューが表示されるので、「**API keys**」（APIキー）をクリックします

**別の方法**: 
- 画面左側のメニューから「**API keys**」を探してクリック

### ステップ4: 新しいAPIキーを作成する

1. 「**+ Create new secret key**」または「**新しいシークレットキーを作成**」ボタンをクリックします
2. 名前を入力する画面が表示されます：
   - 例：「ブログ自動投稿用」など、わかりやすい名前を入力
   - 名前は何でもOKです
3. 「**Create secret key**」または「**作成**」ボタンをクリックします
4. **APIキーが表示**されます（例：`sk-`で始まる長い文字列）
   - ⚠️ **超重要**: このキーも**一度しか表示されません**！
   - 📋 **必ず今すぐコピーしてメモ帳に保存してください**
   - 例: `sk-abcdefghijklmnopqrstuvwxyz1234567890`

### ステップ5: APIキーを保存する

コピーしたAPIキーを`.env`ファイルの`OPENAI_API_KEY`に設定します。

## 📝 パート3: `.env`ファイルを作成する

### ステップ1: プロジェクトフォルダを開く

1. ターミナル（Mac）またはコマンドプロンプト（Windows）を開きます
2. 以下のコマンドを実行して、プロジェクトフォルダに移動します：
   ```bash
   cd "/Users/keiji/Desktop/開発/新規事業（人間関係）"
   ```

### ステップ2: `.env`ファイルを作成する

1. テキストエディタ（メモ帳やVS Codeなど）を開きます
2. 以下の内容をコピー＆ペーストします：

```env
# WordPress設定
WORDPRESS_URL=https://freeeeeeestyle.com
WORDPRESS_USERNAME=ここにあなたのWordPressのユーザー名を入力
WORDPRESS_PASSWORD=ここに取得したアプリケーションパスワードを貼り付け

# OpenAI API設定
OPENAI_API_KEY=ここに取得したOpenAI APIキーを貼り付け

# スケジューラー設定（毎日何時に投稿するか）
POST_TIME=09:00
```

### ステップ3: 情報を入力する

以下のように、実際の値に置き換えてください：

**例：**
```env
# WordPress設定
WORDPRESS_URL=https://freeeeeeestyle.com
WORDPRESS_USERNAME=keiji
WORDPRESS_PASSWORD=abcd efgh ijkl mnop

# OpenAI API設定
OPENAI_API_KEY=sk-abcdefghijklmnopqrstuvwxyz1234567890

# スケジューラー設定
POST_TIME=09:00
```

### ステップ4: ファイルを保存する

1. ファイル名を「`.env`」（ピリオドから始まる）にして保存します
   - ⚠️ **注意**: `.txt`は付けないでください
   - 保存場所は、プロジェクトフォルダ内です

**Macの場合（テキストエディット）:**
- 「フォーマット」→「標準テキストにする」を選択
- 「.env」で保存しようとすると「拡張子なしで保存しますか？」と聞かれるので「拡張子なしを使用」を選択

**Windowsの場合:**
- ファイル名を入力する時に、引用符で囲む: `".env"`
- または「ファイルの種類」で「すべてのファイル」を選択

## ✅ 設定の確認方法

設定が正しくできているか確認するには：

1. ターミナルで以下のコマンドを実行します：
   ```bash
   python main.py
   ```

2. エラーが出ずに記事が投稿されれば成功です！

## 🆘 よくある質問

### Q1: アプリケーションパスワードが見つからない！

**A:** WordPressのバージョンによって、場所が異なる場合があります：
- 「設定」→「一般」→「アプリケーションパスワード」
- または、プラグイン「Application Passwords」をインストールする必要がある場合があります

### Q2: APIキーが見当たらない！

**A:** 
- まず、OpenAIのアカウントにクレジットカードが登録されているか確認してください
- 無料プランでも使えますが、最初にアカウント設定が必要な場合があります
- 「API keys」ページが見つからない場合は、画面左のメニューから探してください

### Q3: `.env`ファイルが保存できない！

**A:**
- Macの場合: ターミナルで `touch .env` と入力してEnterを押し、その後テキストエディタで開く
- Windowsの場合: コマンドプロンプトで `echo. > .env` と入力してEnterを押し、その後メモ帳で開く

### Q4: パスワードやキーを忘れた・消してしまった！

**A:**
- WordPress: プロフィールページで新しいアプリケーションパスワードを作成してください（古いのは削除できます）
- OpenAI: API Keysページで新しいキーを作成してください（古いキーは無効にできます）
